# Vessel v0.1.0-alpha.1

> 首个公开 alpha · First public alpha

**Reflection-driven AI agent harness for Unreal Engine 5.7 development.**
让 Unreal 策划 / 美术 / 程序在 Editor 内用自然语言调用 AI agent —— agent 的每个改动都可预览、可撤销、可审计。

---

## Highlights

- **反射驱动 Tool Registry** — 写一个带 `meta=(AgentTool="true")` 的 `UFUNCTION`,自动导出为 agent tool,无需手写 schema
- **Session Machine** — 8 态显式 FSM,每个状态切换点查 budget,LLM 不做自循环黑箱
- **HITL Gate** — 默认 on,所有写操作必经 Slate 面板审批,Reject 理由自动沉淀到项目 `AGENTS.md`
- **Designer Assistant 默认模板** — 开 Editor 即用,首发支持 DataTable + Asset + Validator 工具集
- **Mock LLM Provider** — 全部 69 个 automation tests 在 CI 上离线跑,不烧真 API key
- **完整三册文档** — 产品 / 工程 / 协作 + 7 条 ADR,任何贡献者都能从文档建立心智模型

## Verification

- ✅ UE 5.7 host-project 编译通过
- ✅ 69 / 69 automation tests 全绿
- ✅ Editor 内 Slate Dock Panel 加载 + 模块启动正常
- ✅ 真 Anthropic 端到端会话(Sonnet 4.6 planner / Haiku 4.5 judge)在真 DataTable 上 round-trip 通过

## Known Limitations(已显式延后到 v0.2)

- **Edit-and-Approve UI** 仍是 disabled 占位
- **Slate widget automation 测试**需 render pump,仅做 module-level smoke
- **Anthropic provider 多轮 tool_use 链**骨架已就位,完整链留 v0.2 打磨
- **Chat 面板渲染最小化** —— LLM reasoning / Plan / Tool result / Judge 评语都落入 `<Project>/Saved/AgentSessions/<id>.jsonl`,但面板只显示 FSM 终态字符串
- **Cost 追踪未实装** —— 面板显示 `$0.00`;`MaxCostUsd` budget 名存实未限,请通过 Anthropic 控制台监控

## Engine Support

- 仅在 **Unreal Engine 5.7** 上端到端验证
- UE 5.5 / 5.6 兼容是 v0.2 目标 · 5.4 及以下不支持

## Quickstart

```bash
cd <YourProject>/Plugins
git clone https://github.com/oopwoof/uevessel.git Vessel
```

然后:Generate VS project files → 编译 → 启动 Editor → 填 API key(Editor Preferences → Vessel) → `Window → Vessel Chat`。

完整文档见 [README](https://github.com/oopwoof/uevessel#readme) 和 [BUILD.md](https://github.com/oopwoof/uevessel/blob/main/docs/engineering/BUILD.md)。
